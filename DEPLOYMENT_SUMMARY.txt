SUPPORT & TROUBLESHOOTING:

See DEPLOYMENT_GUIDE.md for detailed troubleshooting.

Common issues:
  • Route53 zone not found → Create hosted zone in AWS console
  • Connection timeout → Check security groups
  • Instances not starting → Check IAM permissions
  • Docker not running → Check user data logs on EC2

SSH to instance:
  INSTANCE_ID=$(aws autoscaling describe-auto-scaling-groups \
    --auto-scaling-group-names recruitment-asg \
    --query 'AutoScalingGroups[0].Instances[0].InstanceId' \
    --output text)
  
  aws ec2-instance-connect open-tunnel --instance-id $INSTANCE_ID

View logs on instance:
  docker-compose logs -f
